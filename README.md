## Installations instruktioner
1. Ladda ner repo
2. Kontrollera att php ligger i miljövariabler.
3. Kör kommandot ```php composer.phar install``` i rooten för projektet.
4. Öppna upp projektet i localhost.
5. Du kan nu köra applikationen.
